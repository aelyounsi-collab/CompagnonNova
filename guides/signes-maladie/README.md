# Signes de Maladie — CompagnonNova

Guides pour aider les propriétaires à identifier les signes indiquant que leur animal est malade.

## Objectif

Permettre aux propriétaires de :
- Détecter précocement les problèmes de santé
- Comprendre quand une consultation vétérinaire est nécessaire
- Éviter de banaliser des symptômes qui peuvent être graves

## Format recommandé

- Liste numérotée des signes (Top 5, Top 7, Top 10)
- Explication de chaque signe
- Niveau d'urgence pour chaque signe
- Quand appeler le vétérinaire
