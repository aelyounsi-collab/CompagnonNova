# Signes de Douleur — CompagnonNova

Guides pour reconnaître les signes de douleur chez le chien et le chat.

## Particularités

Les animaux dissimulent naturellement leur douleur. Ces guides apprennent aux propriétaires à :
- Décoder le langage corporel de leur animal
- Identifier les changements de comportement suspects
- Agir avant que la douleur devienne invalidante

## Signes universels à couvrir

- Changements de posture
- Modifications du comportement alimentaire
- Vocalises inhabituelles
- Agressivité soudaine
- Réduction de l'activité
- Boiterie
- Léchage excessif d'une zone
