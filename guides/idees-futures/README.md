# Idées Futures — Banque d'idées

Ce dossier stocke les idées de guides qui n'ont pas encore été développées.

## Comment ajouter une idée

Ajouter une ligne dans le fichier thématique correspondant avec :
- Le titre potentiel
- L'angle prévu
- Le potentiel viral (1–5)
