# Guides Chats — CompagnonNova

Guides spécifiques à la santé et au bien-être des chats.

## Spécificités du chat

- Le chat dissimule beaucoup mieux la douleur que le chien
- Certains produits sûrs pour le chien sont mortels pour le chat (ex : huiles essentielles, paracétamol)
- Le chat a des besoins nutritionnels spécifiques (acide aminosulfonique taurine, vitamine A...)

## Thèmes spécifiques chats
- Maladies rénales (très fréquentes)
- Maladies urinaires (obstructions urinéaires)
- Griffes et grattage
- Comportement félin
- Chat d'intérieur vs extérieur
