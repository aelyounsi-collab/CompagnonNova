# Guides — CompagnonNova

Ce dossier centralise tous les guides thématiques produits par CompagnonNova.

## Organisation par thème

| Dossier | Description |
|---------|-------------|
| `urgences-veterinaires/` | Situations d'urgence et conduite à tenir |
| `signes-maladie/` | Identifier les symptômes de maladies |
| `signes-douleur/` | Reconnaître quand son animal souffre |
| `intoxications/` | Produits toxiques et gestes d'urgence |
| `alimentation/` | Ce qu'on peut ou non donner à son animal |
| `dangers-caches/` | Dangers insoupçonnés du quotidien |
| `chats/` | Guides spécifiques aux chats |
| `chiens/` | Guides spécifiques aux chiens |
| `idees-futures/` | Banque d'idées pour les prochains guides |

## Conventions de nommage

Chaque guide suit le format :
```
guides/[theme]/[numero]-[slug-du-titre].md
Ex: guides/urgences-veterinaires/001-chien-ne-respire-plus.md
```

## Structure d'un guide

Se référer à `/templates/template-guide.md`
