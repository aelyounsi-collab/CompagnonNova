# Alimentation — CompagnonNova

Guides sur l'alimentation saine et les aliments dangereux pour chiens et chats.

## Angles éditoriaux

### Ce qu'on peut donner
- Fruits et légumes sûrs pour chiens
- Fruits et légumes sûrs pour chats
- Viandes recommandées
- Suppléments bénéfiques

### Ce qu'on ne peut PAS donner
- Aliments mortels (chocolat, raisin, xylitol...)
- Aliments dangereux à éviter
- Erreurs alimentaires courantes

### Conseils nutritionnels
- Alimentation équilibrée chien / chat
- Croquettes vs alimentation maison
- Alimentation selon l'âge
- Gestion du poids et obésité
