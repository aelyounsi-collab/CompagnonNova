# Urgences Vétérinaires — CompagnonNova

Guides sur la reconnaissance et la gestion des urgences vétérinaires pour chiens et chats.

## Principe

Chaque guide d'urgence doit :
1. Identifier clairement les signes d'urgence
2. Expliquer QUAND aller chez le vétérinaire
3. Donner les gestes d'urgence si applicable
4. TOUJOURS recommander de contacter un vétérinaire

## Mention légale obligatoire

> ⚠️ Ces informations sont à titre éducatif uniquement. En cas de doute ou d'urgence réelle, contactez immédiatement un vétérinaire ou une clinique vétérinaire d'urgence.
