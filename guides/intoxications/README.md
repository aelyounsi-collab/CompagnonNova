# Intoxications — CompagnonNova

Guides sur les intoxications courantes chez le chien et le chat.

## Urgence absolue

Toute suspicion d'intoxication est une urgence vétérinaire. Ces guides doivent :
1. Lister les produits / aliments toxiques
2. Expliquer les symptômes d'intoxication
3. Donner les gestes à faire ET à ne PAS faire
4. Rediriger systématiquement vers le vétérinaire

## Numéros d'urgence à inclure

- **CAPAE-Ouest** (Centre Antipoison Animal) : 02 40 68 77 40
- **CNITV** (Lyon) : 04 78 87 10 40
- **Urgences vétérinaires locales** : toujours consulter

## Thèmes prioritaires
- Chocolat et théobromine
- Raisin et raisins secs
- Xylitol
- Oignons, ail, poireaux
- Paracétamol et ibuprofene
- Rodenticides (mort-aux-rats)
- Plantes toxiques
