# Scripts Vidéo — CompagnonNova

Ce dossier contient tous les scripts de vidéos CompagnonNova.

## Organisation

```
scripts/
├── youtube/        # Scripts longs format
├── tiktok/         # Scripts courts
├── instagram/      # Scripts Reels
├── facebook/       # Scripts vidéos Facebook
└── templates/      # Modèles réutilisables
```

## Conventions

- Nommage : `[numero]-[theme]-[plateforme].md`
- Toujours indiquer la durée estimée
- Inclure les indications de ton et d'émotion
- Marquer les pauses et accents importants
