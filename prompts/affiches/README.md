# Prompts Affiches — CompagnonNova

Prompts pour la génération d'affiches visuelles.

## Outils concernés
- Midjourney
- DALL-E
- Canva (templates)
- Adobe Firefly

## Structure des prompts

Chaque prompt affiche doit spécifier :
- Style : professionnel, moderne, bleu marine et doré
- Personnages : Maely ou Peanut
- Texte à intégrer
- Format : carré, vertical ou horizontal
