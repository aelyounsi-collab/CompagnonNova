# Prompts IA — CompagnonNova

Ce dossier centralise tous les prompts utilisés pour la génération de contenu IA.

## Organisation

| Dossier | Outil | Usage |
|---------|-------|-------|
| `heygen/` | HeyGen | Génération de vidéos avec avatars |
| `hyperframes/` | Hyperframes | Animations et vidéos avancées |
| `claude/` | Claude (Anthropic) | Scripts, guides, stratégie |
| `chatgpt/` | ChatGPT (OpenAI) | Variations, idéation |
| `affiches/` | Midjourney/Canva | Génération d'affiches |
| `miniatures/` | Midjourney/Canva | Génération de miniatures |
| `videos/` | Divers | Prompts vidéo généraux |

## Bonnes pratiques

1. Toujours versionner les prompts (v1, v2...)
2. Documenter les résultats obtenus
3. Indiquer le contexte d'utilisation
4. Partager les prompts qui fonctionnent bien
