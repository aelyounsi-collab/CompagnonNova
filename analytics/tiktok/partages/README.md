# Partages TikTok — CompagnonNova

## Importance des partages

Les partages sont le principal levier de viralité sur TikTok. Ils indiquent que l'utilisateur :
- Veut alerter ses proches
- Identifie quelqu'un dans la situation
- Trouve le contenu suffisamment important pour le diffuser
