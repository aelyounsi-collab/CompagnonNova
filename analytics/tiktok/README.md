# Analytics TikTok — CompagnonNova

## Mission

Suivre, analyser et optimiser les performances du compte TikTok CompagnonNova.

## Structure

```
tiktok/
├── exports/         → Données brutes exportées depuis TikTok Studio
├── rapports/        → Rapports mensuels
├── tops-videos/     → Meilleures vidéos documentées
├── engagement/      → Analyse de l'engagement (likes, commentaires)
├── favoris/         → Analyse des favoris (sauvegardes)
├── partages/        → Analyse des partages
├── abonnes/         → Évolution des abonnés
└── recommandations/ → Actions concrètes
```

## KPI TikTok principaux

| Métrique | Description | Seuil cible |
|---------|-------------|-------------|
| Taux de complétion | % de la vidéo regardée | > 60% |
| Taux de partage | Partages / Vues | > 2% |
| Taux de favori | Favoris / Vues | > 3% |
| Abonnés générés / vidéo | | > 10 |
| Vues moyennes | Par vidéo | > 1 000 |
