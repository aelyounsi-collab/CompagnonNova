# Favoris TikTok — CompagnonNova

## Importance des favoris

Les favoris (sauvegardes) sur TikTok indiquent que l'utilisateur :
- Veut revenir au contenu
- Juge le contenu utile à conserver
- Partage en privé

C'est un signal fort de **valeur utilitaire** du contenu.
