# Engagement TikTok — CompagnonNova

## Métriques d'engagement

| Métrique | Description | Signal |
|---------|-------------|--------|
| Likes | Appréciation passive | Bon contenu |
| Commentaires | Réaction active | Fort engagement |
| Partages | Diffusion | Contenu à haute valeur |
| Favoris | Sauvegarde | Contenu utile / référence |
| Duos/Stitch | Reprise | Viralité potentielle |

## Analyse des commentaires

Types de commentaires à tracker :
- Questions (opportunités de contenu)
- Témoignages (preuves sociales)
- Objections (axes d'amélioration)
- Partages d'expérience
