# Sauvegardes Instagram — CompagnonNova

## Pourquoi les sauvegardes sont la métrique reine

Une sauvegarde signifie que l'utilisateur :
- Juge le contenu suffisamment utile pour le retrouver
- Fait confiance à la source
- Reviendra sur ce contenu

C'est le signal le plus fort de **valeur perçue**.
