# Analytics Instagram — CompagnonNova

## Mission

Suivre, analyser et optimiser les performances du compte Instagram CompagnonNova.

## Structure

```
instagram/
├── exports/         → Données brutes depuis Meta Business Suite
├── rapports/        → Rapports mensuels
├── reels/           → Analyse spécifique des Reels
├── sauvegardes/     → Analyse des sauvegardes (signal fort)
├── partages/        → Analyse des partages
├── abonnes/         → Évolution des abonnés
└── recommandations/ → Actions concrètes
```

## KPI Instagram principaux

| Métrique | Description | Seuil cible |
|---------|-------------|-------------|
| Sauvegardes / post | Signal d'utilité | > 20 |
| Taux complétion Reels | % visionnage | > 50% |
| Portée | Comptes touchés | Croissance |
| Clics lien bio | Trafic sortant | > 50/mois |
| Taux engagement | (Likes+Comments+Save)/Reach | > 5% |
