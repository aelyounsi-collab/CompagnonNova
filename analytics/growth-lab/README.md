# Growth Lab — Analytics CompagnonNova

> Laboratoire de croissance : hypothèses, tests, conclusions et roadmap basés sur les données réelles.

## Structure

```
growth-lab/
├── hypotheses/                  → Hypothèses à tester
├── tests/                       → Tests en cours
├── conclusions/                 → Résultats validés
├── succes/                      → Ce qui fonctionne (à reproduire)
├── echecs/                      → Ce qui ne fonctionne pas (à éviter)
├── roadmap/                     → Feuille de route basée sur les données
└── recommandations-globales/    → Synthèse cross-plateformes
```

## Processus

```
Idée → Hypothèse → Test → Mesure → Conclusion → Action
```

## Règles du Growth Lab

1. **Toute décision stratégique** doit être basée sur des données
2. **Une hypothèse à la fois** par plateforme
3. **Durée minimum d'un test** : 2 semaines / 5 publications
4. **Documenter tout** : même les échecs apprennent
5. **Pas de vanity metrics** : vues sans engagement = rien
