## Hypothèses en file d'attente

### H001 — Miniatures avec Peanut = CTR plus élevé
- **Plateforme** : YouTube
- **Hypothèse** : Si Peanut apparaît dans la miniature, alors le CTR augmente de +20%, parce que les animaux attirent l'attention
- **Méthode** : Alterner miniatures avec/sans Peanut sur 10 vidéos
- **Statut** : À tester

### H002 — Hooks "STOP !" génèrent plus de complétion
- **Plateforme** : TikTok
- **Hypothèse** : Les vidéos commençant par "STOP !" ont un taux de complétion +15%
- **Méthode** : Comparer 5 vidéos avec vs 5 sans ce hook
- **Statut** : À tester

### H003 — Publication 18h > 12h sur TikTok
- **Plateforme** : TikTok
- **Hypothèse** : Publier à 18h génère +30% de vues initiales vs 12h
- **Méthode** : Publier en alternance sur 4 semaines
- **Statut** : À tester
