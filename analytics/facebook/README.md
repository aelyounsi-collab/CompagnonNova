# Analytics Facebook — CompagnonNova

## Mission

Suivre, analyser et optimiser les performances de la page Facebook CompagnonNova.

## Structure

```
facebook/
├── exports/         → Données brutes depuis Meta Business Suite
├── rapports/        → Rapports mensuels
├── reels/           → Analyse des Reels Facebook
├── portee/          → Analyse de la portée (organique vs virale)
├── engagement/      → Analyse de l'engagement
├── abonnes/         → Évolution des fans
└── recommandations/ → Actions concrètes
```

## KPI Facebook principaux

| Métrique | Description | Seuil cible |
|---------|-------------|-------------|
| Portée organique | Comptes touchés sans pub | Croissance |
| Portée virale | Via partages | > 30% portée totale |
| Taux engagement | Interactions / Portée | > 3% |
| Partages | Principal levier viral | > 10/post |
| Vues vidéos | > 3 secondes | > 500/vidéo |
