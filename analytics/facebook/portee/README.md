# Portée — Facebook CompagnonNova

## Types de portée

| Type | Description | Levier d'action |
|------|-------------|------------------|
| Organique | Vus par les fans | Constance + horaires |
| Virale | Vus via partages | Contenu partageable |
| Payante | Publicité | Budget ads |

## Facteurs qui augmentent la portée organique
- Publications régulières
- Contenu natif (pas de liens sortants dans le post)
- Vidéos uploadées directement
- Engagement rapide après publication
