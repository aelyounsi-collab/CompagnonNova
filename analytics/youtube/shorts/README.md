# YouTube Shorts — CompagnonNova

## Stratégie Shorts

Les Shorts permettent :
- D'atteindre de nouveaux spectateurs via le feed Shorts
- De réutiliser des extraits de vidéos longues
- De tester des hooks et formats rapidement

## KPI Shorts

| Métrique | Seuil cible |
|---------|-------------|
| Vues | > 1 000/Short |
| Taux de complétion | > 70% |
| Abonnés générés | > 2/Short |
