# Analytics YouTube — CompagnonNova

## Mission

Suivre, analyser et optimiser les performances de la chaîne YouTube CompagnonNova.

## Structure

```
youtube/
├── exports/            → Fichiers CSV/JSON exportés depuis YouTube Studio
├── rapports/           → Rapports mensuels d'analyse
├── tops-videos/        → Meilleures vidéos documentées
├── mots-cles/          → Analyse des mots-clés et opportunités SEO
├── retention/          → Courbes et analyses de rétention
├── trafic/             → Sources de trafic et comportement audience
├── abonnes/            → Évolution et analyse des abonnés
├── shorts/             → Analytics spécifiques aux Shorts
└── recommandations/    → Actions concrètes issues des analyses
```

## KPI YouTube principaux

| Métrique | Description | Seuil cible |
|---------|-------------|-------------|
| CTR miniature | % clics / impressions | > 5% |
| Rétention moyenne | % de la vidéo regardée | > 50% |
| Watch time mensuel | Heures de visionnage | Croissance +10%/mois |
| Abonnés générés | Par vidéo | > 5 |
| Impressions | Vues dans les suggestions | Croissance |
