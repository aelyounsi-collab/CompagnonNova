# Workflow Growth — CompagnonNova

> Processus de croissance et d'optimisation continue.

---

## Revue de croissance mensuelle

1. Analyser les KPI par rapport aux objectifs (voir `ROADMAP.md`)
2. Identifier les leviers de croissance actifs
3. Tester de nouveaux angles ou formats
4. Documenter les hypothèses dans `docs/growth-lab.md`
5. Prioriser les actions pour le mois suivant

## Tests à mener systématiquement

### Tests de miniatures
- Tester 2 versions de miniatures sur les nouvelles vidéos
- Comparer le CTR après 48h
- Garder la gagnante

### Tests de titres
- Format question vs affirmation
- Avec chiffre vs sans
- Avec émotion vs informatif

### Tests de formats
- Durée : 30s vs 60s vs 90s (TikTok)
- Fond : studio vs extérieur
- Maely seule vs avec Peanut

## Acquisition

- Collaborations avec d'autres comptes animaux
- Partage croisé entre plateformes
- Contenu "partagé par" et challenges
- Réponses aux commentaires en vidéo
