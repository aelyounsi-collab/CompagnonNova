# Workflow Publication — CompagnonNova

> Processus de publication multi-plateformes.

---

## Ordre de publication recommandé

1. **YouTube** (version longue) — priorité 1
2. **TikTok** (version courte) — 1–2h après YouTube
3. **Instagram** (Reel) — simultanément avec TikTok
4. **Facebook** (vidéo native) — le lendemain matin

## Préparation

- [ ] Pack de publication complet prêt (`packs-publication/`)
- [ ] Vidéo dans tous les formats (16:9 + 9:16)
- [ ] Miniatures créées
- [ ] Hashtags sélectionnés
- [ ] Horaires programmés

## Publication YouTube

- [ ] Uploader la vidéo (1080p minimum)
- [ ] Ajouter le titre, la description, les tags
- [ ] Ajouter la miniature
- [ ] Assigner aux playlists
- [ ] Ajouter les chapitres
- [ ] Ajouter les cartes et écran de fin
- [ ] Publier à l'heure programmée
- [ ] Épingler le commentaire

## Publication TikTok

- [ ] Uploader la version 9:16
- [ ] Ajouter la caption et les hashtags
- [ ] Choisir la musique
- [ ] Publier

## Publication Instagram

- [ ] Uploader en tant que Reel
- [ ] Rédiger la caption
- [ ] Ajouter les hashtags
- [ ] Créer la story associée
- [ ] Publier

## Publication Facebook

- [ ] Uploader la vidéo nativement
- [ ] Rédiger le texte du post
- [ ] Publier (lien YouTube en commentaire, pas dans le post)
- [ ] Partager dans le groupe communautaire

## Suivi post-publication

- [ ] Répondre aux commentaires (toutes plateformes) dans les 2h
- [ ] Surveiller les stats à 24h
- [ ] Partager les extraits si la vidéo performe bien
