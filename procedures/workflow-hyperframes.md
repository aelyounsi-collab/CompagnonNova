# Workflow Hyperframes — CompagnonNova

> Processus de production vidéo avec Hyperframes.

---

## Usage d'Hyperframes

Hyperframes est utilisé pour :
- Animations avancées
- Formats vidéo alternatifs
- Effets visuels spéciaux
- Complément à HeyGen pour certains formats

## Processus

1. Définir le besoin (animation spécifique, effet particulier)
2. Consulter les prompts dans `prompts/hyperframes/`
3. Générer et itérer
4. Exporter et intégrer dans le workflow standard
5. Documenter le prompt utilisé si le résultat est bon
