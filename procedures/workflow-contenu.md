# Workflow Contenu — CompagnonNova

> Processus complet de création de contenu, de l'idée à la publication.

---

## Étape 1 — Idéation

**Responsable** : Équipe stratégie  
**Outil** : `guides/idees-futures/`, `guides/priorites.md`

- Consulter la banque d'idées
- Vérifier le calendrier éditorial
- Choisir le sujet en fonction des priorités et de la saisonnalité
- Valider l'angle et le format

**Output** : Sujet validé + format + plateforme cible

---

## Étape 2 — Recherche & Validation

**Responsable** : Rédacteur  
**Sources** : Sources vétérinaires fiables (Vétofocus, revues scientifiques, vétérinaires reconnus)

- Vérifier les informations sur des sources fiables
- S'assurer qu'aucune information n'est incorrecte ou dangereuse
- Ajouter les numéros d'urgence si nécessaire

**Output** : Notes de recherche validées

---

## Étape 3 — Script / Guide

**Responsable** : Rédacteur + IA (Claude)  
**Template** : `scripts/templates/` ou `templates/template-guide.md`

- Rédiger le script ou le guide avec le template correspondant
- Vérifier la longueur et le ton
- Inclure la mention légale
- Relire et corriger

**Output** : Script ou guide final

---

## Étape 4 — Production Visuelle

**Responsable** : Graphiste / IA  
**Outils** : HeyGen, Canva, Midjourney

- Créer la vidéo (HeyGen avec Maely)
- Créer l'affiche
- Créer la miniature
- Exporter dans tous les formats nécessaires

**Output** : Vidéo + affiche + miniature finalisés

---

## Étape 5 — Pack de Publication

**Responsable** : Rédacteur  
**Template** : `templates/template-pack.md`

- Rédiger le titre pour chaque plateforme
- Rédiger la description
- Sélectionner les hashtags
- Préparer le commentaire épinglé
- Finaliser le CTA

**Output** : Pack de publication complet dans `packs-publication/`

---

## Étape 6 — Publication

**Responsable** : Community Manager  
**Checklist** : Voir `plateformes/[plateforme]/checklist-publication.md`

- Publier à l'horaire optimal
- Ajouter les éléments spécifiques à la plateforme
- Surveiller les premières 2 heures
- Répondre aux commentaires

**Output** : Contenu publié et suivi

---

## Étape 7 — Analyse

**Responsable** : Analyste  
**Délai** : 7 jours et 30 jours après publication  
**Fichier** : `analytics/[plateforme]/`

- Recueillir les statistiques
- Identifier les points forts et faibles
- Documenter les enseignements
- Adapter la stratégie si nécessaire

**Output** : Rapport dans `analytics/`
