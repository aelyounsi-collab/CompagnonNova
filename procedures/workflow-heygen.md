# Workflow HeyGen — CompagnonNova

> Processus de production vidéo avec HeyGen et l'avatar Maely.

---

## Pré-production

1. **Script final** disponible dans `scripts/`
2. Vérifier le ton et la durée estimée
3. Identifier la variante de Maely nécessaire
4. Préparer la scène HeyGen (fond, éclairage)

## Production

1. Se connecter à HeyGen
2. Sélectionner l'avatar Maely (configuration dans `prompts/heygen/avatar-config.md`)
3. Coller le script dans le générateur
4. Ajuster la vitesse de parole si nécessaire
5. Lancer la génération
6. Vérifier le résultat (lipsync, expression, fluidité)

## Post-production

1. Télécharger la vidéo générée
2. Ajouter l'intro et l'outro CompagnonNova
3. Ajouter les sous-titres
4. Ajouter le logo en filigrane
5. Ajouter la musique de fond (niveau : fond sonore discret)
6. Exporter dans les formats nécessaires
7. Créer les versions courtes pour TikTok/Instagram depuis la version longue

## Contrôle qualité

- [ ] Lipsync correct
- [ ] Expression de Maely adaptée au ton
- [ ] Sous-titres corrects
- [ ] Musique non glanante
- [ ] Intro/Outro présents
- [ ] Logo visible
- [ ] Mention légale incluse
