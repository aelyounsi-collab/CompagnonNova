# Automatisations — CompagnonNova

Ce dossier documente les automatisations mises en place et planifiées pour CompagnonNova.

## Outils d'automatisation

| Dossier | Outil | Usage |
|---------|-------|-------|
| `n8n/` | n8n | Automatisations open-source |
| `make/` | Make (ex-Integromat) | Automatisations no-code |
| `zapier/` | Zapier | Intégrations rapides |
| `ia/` | IA diverses | Génération automatisée |

- Production et publication manuelles
- Analytiques manuelles

- [ ] Génération de packs de publication automatisée

### Phase 3 (automatisée)
- [ ] Publication automatique programmée
- [ ] Rapport analytics automatisé (mensuel)
