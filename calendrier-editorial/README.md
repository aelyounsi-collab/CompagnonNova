# Calendrier Éditorial — CompagnonNova

Ce dossier contient la planification du contenu par mois, par thème et par plateforme.

## Organisation

- Un fichier par mois : `YYYY-MM.md`
- Vue globale par trimestre
- Suivi du statut de chaque contenu

## Statuts

| Statut | Description |
|--------|-------------|
| 🟡 Idée | Sujet identifié, pas encore traité |
| 🔵 En cours | Script ou guide en cours de rédaction |
| 🟠 Production | En cours de production vidéo/visuel |
| 🟢 Prêt | Pack complet, prêt à publier |
| ✅ Publié | Contenu publié sur les plateformes |
| 📊 Analysé | Performance analysée et documentée |

## Cadence cible

- **YouTube** : 2 vidéos/semaine
- **TikTok** : 5 vidéos/semaine
- **Instagram** : 4 posts/semaine
- **Facebook** : 3 posts/semaine
