# Archives — CompagnonNova

Ce dossier contient les contenus archivés, les anciennes versions et les éléments retirés de la production active.

## Règles d'archivage

- Un contenu est archivé s'il est obsolète ou remplacé par une version améliorée
- Conserver la date d'archivage dans le nom du fichier
- Ne jamais supprimer définitivement sans validation
