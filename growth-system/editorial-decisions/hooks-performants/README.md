# Hooks Performants — CompagnonNova

> Hooks validés par les données. À systématiser dans les scripts.

## Définition

Un hook est l'accroche des 3 premières secondes d'une vidéo. C'est l'élément le plus déterminant pour le taux de complétion sur TikTok et la rétention sur YouTube.

## Hooks validés

### Formule 1 — STOP / Alerte
```
"STOP ! Si ton [chien/chat] fait ça, c'est une urgence."
```

- **Plateformes** : TikTok, Instagram Reels
