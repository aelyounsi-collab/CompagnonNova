# Décisions Éditoriales — CompagnonNova

> Ce dossier centralise toutes les décisions éditoriales prises sur la base des données.

Chaque décision est datée et justifiée par un signal mesuré.

## Structure

```
editorial-decisions/
├── sujets-a-reproduire/    → Sujets validés par les données
├── sujets-a-eviter/        → Sujets qui ne fonctionnent pas
├── hooks-performants/      → Hooks validés à systématiser
├── cta-performants/        → CTA qui convertissent
└── formats-performants/    → Formats à privilégier
```

## Règle

Un élément est ajouté ici uniquement après **minimum 3 occurrences** validant le pattern.
