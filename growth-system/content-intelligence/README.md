# Content Intelligence — CompagnonNova

> Base de connaissance des patterns de performance. La mémoire éditoriale de CompagnonNova.

## Mission

Conserver et organiser tout ce qui a été appris sur la performance des contenus. Chaque pattern validé par les données est documenté ici pour être reproductible.

## Structure

```
content-intelligence/
├── meilleurs-sujets/      → Sujets à fort potentiel
├── meilleurs-hooks/       → Accroches les plus efficaces
├── meilleurs-cta/         → Appels à l'action qui convertissent
├── meilleures-miniatures/  → Patterns de miniatures performantes
├── meilleures-affiches/   → Patterns d'affiches performantes
└── meilleures-videos/     → Analyse des vidéos les plus performantes
```

## Principe

Ce n'est pas un archive — c'est un **playbook vivant**. Chaque entrée est utilisée activement dans la production.
