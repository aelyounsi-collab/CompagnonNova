# Couleurs Officielles — CompagnonNova

## Palette principale

### Bleu Marine Profond
- **Hex** : `#0B0F1A`
- **RGB** : 11, 15, 26
- **Usage** : Fond principal, barres de titre, éléments d'autorité
- **Symbolique** : Profondeur, sérieux, expertise

### Doré Élégant
- **Hex** : `#C9A84C`
- **RGB** : 201, 168, 76
- **Usage** : Accents, titres, éléments premium, CTA
- **Symbolique** : Excellence, qualité, confiance

### Vert CompagnonNova
- **Hex** : `#2D6A4F`
- **RGB** : 45, 106, 79
- **Usage** : Santé, bien-être, messages positifs
- **Symbolique** : Nature, vie, santé

### Blanc
- **Hex** : `#FFFFFF`
- **Usage** : Textes sur fond sombre, clarté

## Palette secondaire

- **Gris anthracite** `#2C2C2C` — textes secondaires
- **Rouge urgence** `#C0392B` — alertes uniquement
- **Bleu info** `#3498DB` — informations, conseils

## Combinaisons recommandées

| Fond | Texte | Usage |
|------|-------|-------|
| `#0B0F1A` | `#FFFFFF` | Affiches principales |
| `#0B0F1A` | `#C9A84C` | Titres premium |
| `#FFFFFF` | `#0B0F1A` | Documents, guides |
| `#2D6A4F` | `#FFFFFF` | Badges santé |
