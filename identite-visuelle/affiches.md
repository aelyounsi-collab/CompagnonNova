# Affiches — Spécifications

## Formats standards

| Format | Dimensions | Usage |
|--------|------------|-------|
| Carré | 1080 x 1080 px | Instagram, Facebook |
| Vertical | 1080 x 1920 px | Stories, TikTok |
| Horizontal | 1920 x 1080 px | YouTube, Facebook |

## Structure type

```
+------------------------------------------+
|  LOGO COMPAGNONNOVA                      |
|  [IMAGE PRINCIPALE : Maely / Peanut]     |
|  TITRE ACCROCHEUR EN MAJUSCULES          |
|  [CTA] Regarde la vidéo complète        |
+------------------------------------------+
```

## Éléments obligatoires

- [ ] Logo CompagnonNova visible
- [ ] Titre lisible en moins de 3 secondes
- [ ] Image Maely ou Peanut
- [ ] Couleurs officielles
- [ ] CTA présent
