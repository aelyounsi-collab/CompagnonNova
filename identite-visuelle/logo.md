# Logo — CompagnonNova

## Versions

| Version | Usage | Fond |
|---------|-------|------|
| Principale | Communications officielles | Fond sombre |
| Inversée | Documents clairs | Fond blanc |
| Monochrome | Impression N&B | Tous fonds |
| Favicon | Réseaux sociaux | Carré |

## Règles d'utilisation

- Espace minimum : équivalent hauteur lettre "C"
- Ne jamais étirer ou déformer
- Ne jamais changer les couleurs officielles
- Ne jamais ajouter d'effets (ombre, glow)

## Tailles minimales

- Numérique : 120px largeur
- Impression : 30mm largeur
