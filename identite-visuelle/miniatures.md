# Miniatures Vidéo — Spécifications

## YouTube
- **Dimensions** : 1920 x 1080 px
- **Poids max** : 2 MB
- **Format** : JPG, PNG

### Bonnes pratiques
- Expressions faciales fortes (surprise, inquiétude)
- Texte en gras, 3 mots maximum
- Contraste très fort
- Tester en petit format mobile

## TikTok / Reels
- **Dimensions** : 1080 x 1920 px
- Première seconde = miniature
- Optimiser les 3 premières secondes

## Checklist
- [ ] Lisible en vignette
- [ ] Émotion forte visible
- [ ] Texte < 4 mots, gras
- [ ] Logo discret présent
- [ ] Cohérence avec le titre
