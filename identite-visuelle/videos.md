# Charte Vidéo — CompagnonNova

## Structure type

```
[INTRO] 3-5 sec — Logo animé
[ACCROCHE] 5-10 sec — Question ou fait choc
[DÉVELOPPEMENT] Variable — Information clé
[CTA] 5-10 sec — Appel à l'action
[OUTRO] 3-5 sec — Logo + abonnement
```

## Éléments obligatoires
- Logo en filigrane (coin supérieur droit, 60% opacité)
- Sous-titres sur toutes les vidéos
- Mention légale en description

## Tons selon le contenu

| Type | Ton | Rythme |
|------|-----|--------|
| Urgence | Sérieux, alerte | Rapide |
| Prévention | Pédagogique | Modéré |
| Alimentation | Informatif | Lent-modéré |
| Danger | Alerte | Rapide |
| Bien-être | Chaleureux | Lent |

## Mention légale

> "Ce contenu est à titre informatif uniquement. En cas d'urgence, contactez immédiatement votre vétérinaire."
