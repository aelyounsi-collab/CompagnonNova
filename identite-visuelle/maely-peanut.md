# Maely & Peanut — Personnages Officiels

## Maely — Experte Officielle

### Identité
- **Nom** : Maely
- **Rôle** : Experte officielle, animatrice principale
- **Caractère** : Bienveillante, experte, rassurante, pédagogique
- **Ton** : Professionnel, chaleureux, accessible

### Rôle dans le contenu
- Présente les guides et informations
- Explique les situations d'urgence avec calme
- Rassure les propriétaires inquiets
- Incarne la voix d'autorité de CompagnonNova

### Variantes
- Maely standard (fond bleu marine)
- Maely urgence (ambiance tendue)
- Maely conseil (ton détendu)
- Maely extérieur

### Identité
- **Nom** : Peanut
- **Race** : Cavalier King Charles
