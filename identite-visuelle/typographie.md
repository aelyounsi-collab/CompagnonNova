# Typographie — CompagnonNova

## Polices officielles

### Titres — Montserrat Bold
- Poids : 700 / 800
- Usage : Titres, accroches, CTA
- Source : Google Fonts

### Corps de texte — Open Sans Regular
- Poids : 400 / 600
- Usage : Descriptions, listes, corps
- Source : Google Fonts

### Citations — Playfair Display Italic
- Usage : Citations, sous-titres élégants

## Hiérarchie

| Élément | Police | Taille | Couleur |
|---------|--------|--------|---------|
| Titre principal | Montserrat ExtraBold | 48–72px | `#C9A84C` |
| Titre secondaire | Montserrat Bold | 32–48px | `#FFFFFF` |
| Sous-titre | Montserrat SemiBold | 24–32px | `#FFFFFF` |
| Corps | Open Sans Regular | 16–18px | `#FFFFFF` |
| Légende | Open Sans Regular | 12–14px | `#AAAAAA` |

## Règles

1. Maximum 2 polices par visuel
2. MAJUSCULES pour les titres courts
3. Intéerlignage minimum 1.4
4. Taille minimum 12px sur mobile
