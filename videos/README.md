# Vidéos — CompagnonNova

Ce dossier référence et organise les ressources vidéo de la marque.

## Structure

```
videos/
├── youtube/           # Vidéos longues format YouTube
├── tiktok/            # Vidéos courtes TikTok
├── instagram/         # Reels Instagram
├── facebook/          # Vidéos Facebook
├── branding/          # Intros, outros, jingles
└── raw/               # Fichiers bruts non montés
```

## Formats cibles

| Plateforme | Format | Durée recommandée |
|------------|--------|-------------------|
| YouTube | 16:9 1080p | 3–10 min |
| TikTok | 9:16 1080p | 30–90 sec |
| Instagram Reels | 9:16 1080p | 15–60 sec |
| Facebook | 16:9 ou 9:16 | 1–3 min |
