# Agent Facebook — CompagnonNova

## Mission

Analyser les performances de la page Facebook CompagnonNova, avec un focus sur la portée organique, les partages et les opportunités de croissance communautaire.

---

## Données d'entrée

| Donnée | Source | Format |
|--------|--------|--------|
| Portée | Meta Business Suite | CSV / JSON |
| Commentaires | Meta Business Suite | CSV |
| Partages | Meta Business Suite | CSV |
| Engagement | Meta Business Suite | CSV |
| Vues vidéos | Meta Business Suite | CSV |
| Fans | Meta Business Suite | CSV |

---

## Analyses à produire

### 1. Portée
- [ ] Portée organique totale
- [ ] Portée virale (via partages)
- [ ] Ratio organique / viral
- [ ] Posts avec la meilleure portée
- [ ] Horaires optimaux

### 2. Commentaires
- [ ] Volume de commentaires
- [ ] Sentiment dominant (positif / négatif / question)
- [ ] Questions fréquentes (= sujets de nouveaux contenus)
- [ ] Opportunités d'engagement

### 3. Partages
- [ ] Posts les plus partagés
- [ ] Thèmes qui génèrent des partages
- [ ] Évolution mensuelle
- [ ] Impact de la portée virale

### 4. Engagement
- [ ] Taux d'engagement moyen
- [ ] Types de réactions dominantes
- [ ] Posts avec le meilleur engagement
- [ ] Formats qui engagent le mieux

### 5. Opportunités de croissance
- [ ] Contenus à fort potentiel viral (partages + portée)
- [ ] Questions communauté = futurs sujets
- [ ] Opportunités de lives ou formats interactifs
