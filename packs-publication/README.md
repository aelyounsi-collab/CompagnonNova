# Packs de Publication — CompagnonNova

Ce dossier contient les packs de publication complets, prêts à l'emploi pour chaque plateforme.

## Structure d'un pack

Chaque pack contient :
- **Titre** : Optimisé pour la plateforme
- **Description** : Corps du texte avec mots-clés
- **Hashtags** : Sélection optimale
- **Tags** (YouTube) : 10–20 tags SEO
- **Commentaire épinglé** : Prêt à copier-coller
- **CTA** : Appels à l'action adaptés

## Organisation

```
packs-publication/
├── youtube/
├── tiktok/
├── instagram/
└── facebook/
```

Chaque guide doit avoir son pack dans chacun des dossiers plateforme.

Nommage : `[numero]-[slug-guide]-pack.md`
