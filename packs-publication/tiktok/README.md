# Packs TikTok — CompagnonNova

Packs de publication optimisés pour TikTok.

## Structure type d'un pack TikTok

```markdown
## Caption (150 caractères max visibles)
[Texte accrocheur + emoji]

## Hashtags (3-5)
#CompagnonNova #[hashtag] #[hashtag]

## Son / Musique
[Indication de la bande son]

## CTA dans la vidéo
[Texte du CTA à dire ou afficher]
```
