# Checklist Publication YouTube — CompagnonNova

## Avant la publication

### Script & Contenu
- [ ] Script relu et validé
- [ ] Information vérifiée et fiable
- [ ] Mention légale incluse
- [ ] CTA clairement formulé

### Production vidéo
- [ ] Vidéo exportée en 1080p minimum
- [ ] Sous-titres intégrés ou fichier SRT prêt
- [ ] Intro CompagnonNova présente
- [ ] Outro avec appel à l'abonnement
- [ ] Logo en filigrane visible

### Visuels
- [ ] Miniature créée (1920x1080px)
- [ ] Miniature testée en petit format
- [ ] Cohrence miniature / titre

### Pack de publication
- [ ] Titre optimisé SEO (< 60 caractères)
- [ ] Description rédigée (150+ mots)
- [ ] Tags saisis (10–20)
- [ ] Commentaire épinglé préparé
- [ ] Playlist sélectionnée
- [ ] Chapitres (timestamps) ajoutés

## À la publication
- [ ] Publier à l'horaire optimal
- [ ] Cartes ajoutées (vidéos associées)
- [ ] Écran de fin configuré
- [ ] Partage réseaux effectué

## Après la publication (48h)
- [ ] Répondre aux commentaires
- [ ] Vérifier les stats initiales
- [ ] Extraits pour TikTok/Instagram créés
