# Stratégie YouTube — CompagnonNova

## Positionnement

YouTube est la plateforme principale pour les **guides complets et approfondis** sur la santé animale. C'est le pilier de crédibilité et d'autorité de CompagnonNova.

## Objectifs

- Établir CompagnonNova comme référence en santé animale francophone
- Générer un trafic organique durable via le SEO YouTube
- Convertir les spectateurs en abonnés fidèles
- Alimenter les autres plateformes (extraits TikTok/Instagram)

## Types de contenus

| Type | Fréquence | Durée | Priorité |
|------|-----------|-------|-----------|
| Guides urgences vét. | 2x/semaine | 5–10 min | Haute |
| Signes de maladie | 2x/semaine | 3–8 min | Haute |
| Alimentation | 1x/semaine | 4–7 min | Moyenne |
| Dangers cachés | 1x/semaine | 3–6 min | Moyenne |
| Prévention | 1x/semaine | 4–8 min | Moyenne |

## SEO YouTube

- Titre : mot-clé principal en début de titre
- Description : 150 mots minimum, mots-clés naturels
- Tags : 10–20 tags pertinents
- Chapitres (timestamps) : toujours pour les vidéos > 3 min
- Cartes et écrans de fin : rediriger vers d'autres vidéos

## Cadence de publication

- **Fréquence cible** : 4–5 vidéos/semaine
- **Horaire optimal** : Mardi, Jeudi, Samedi — 18h–20h
- **Constance** avant tout : un calendrier tenu > un pic puis silence
