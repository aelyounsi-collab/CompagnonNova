# Stratégie Instagram — CompagnonNova

## Positionnement

Instagram est la vitrine **estthétique et communautaire** de CompagnonNova. Mise sur les Reels pour la croissance et les stories pour l'engagement quotidien.

## Objectifs
- Image de marque premium
- Engagement de la communauté
- Redirection vers YouTube
- Croissance via les Reels

## Mix de contenu

| Type | Fréquence |
|------|----------|
| Reels (extraits vidéos) | 4x/semaine |
| Affiches informatiques | 2x/semaine |
| Stories (coulisses, sondages) | Quotidien |
| Carrousels (top listes) | 1x/semaine |

## Cadence
- **Publication** : 1 post/jour + stories
- **Horaire optimal** : 12h ou 20h
