# Checklist Publication Instagram — CompagnonNova

## Reels
- [ ] Vidéo 9:16, 1080x1920px
- [ ] Durée 15–60 secondes
- [ ] Sous-titres intégrés
- [ ] Première seconde visuelle forte
- [ ] Audio adapté

## Posts / Affiches
- [ ] Format 1080x1080px ou 1080x1350px
- [ ] Texte lisible sur mobile
- [ ] Couleurs officielles
- [ ] Logo visible

## Caption
- [ ] Accroche en 1ère ligne (visible sans "voir plus")
- [ ] Corps informatif
- [ ] CTA clair
- [ ] 5–10 hashtags pertinents
- [ ] Emoji utilisés avec modération

## Après publication
- [ ] Story créée avec le post
- [ ] Répondre aux commentaires rapidement
