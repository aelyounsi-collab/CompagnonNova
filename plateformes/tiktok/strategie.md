# Stratégie TikTok — CompagnonNova

## Positionnement

TikTok est la plateforme de **croissance rapide et virale**. L'objectif est d'atteindre un maximum de propriétaires d'animaux avec des contenus courts, chocs et partageables.

## Objectifs

- Viraliser les informations clés sur la santé animale
- Attirer de nouveaux abonnés jeunes (18–35 ans)
- Redériver le trafic vers YouTube
- Créer une communauté engagée

## Types de contenus

| Type | Durée | Fréquence |
|------|-------|-----------|
| "Tu savais que..." | 15–30 sec | 2x/semaine |
| Signes d'urgence | 30–60 sec | 2x/semaine |
| Aliments dangereux | 15–45 sec | 2x/semaine |
| Checklist rapide | 30–60 sec | 1x/semaine |

## Format gagnant TikTok

```
[0-3 sec] : Accroche CHOC — "Stop ! Si ton chien fait ça..."
[3-20 sec] : Information rapide et claire
[20-30 sec] : CTA — "Sauvegarde si tu as un animal"
```

## Cadence
- **Fréquence cible** : 5–7 vidéos/semaine
- **Horaire optimal** : 12h ou 18h–21h
