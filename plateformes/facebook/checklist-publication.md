# Checklist Publication Facebook — CompagnonNova

## Posts vidéo
- [ ] Vidéo uploader nativement sur Facebook (meilleure portée)
- [ ] Miniature personnalisée
- [ ] Sous-titres intégrés
- [ ] Description complète avec CTA

## Posts image / affiche
- [ ] Format 1200x630px ou carré
- [ ] Texte < 20% de l'image
- [ ] Description engageante

## Caption
- [ ] Accroche forte en 1ère ligne
- [ ] Corps informatif
- [ ] Question pour favoriser l'engagement
- [ ] CTA (partager, commenter, lien)
- [ ] 2–3 hashtags maximum

## Après publication
- [ ] Répondre aux commentaires dans les 2h
- [ ] Partager dans le groupe communautaire
- [ ] Partager sur le profil personnel (si pertinent)
