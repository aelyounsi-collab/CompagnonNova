# Knowledge Base — CompagnonNova

> Base de connaissance issue de la communauté. Chaque commentaire intéressant, chaque question posée, chaque témoignage d'abonné est une opportunité de contenu.

---

## Mission

Captures les signaux de la communauté pour :
1. Identifier les besoins réels des propriétaires d'animaux
2. Générer des idées de guides et de vidéos
3. Comprendre les inquiétudes les plus fréquentes
4. Valider les sujets à produire en priorité

---

## Structure

```
knowledge-base/
├── commentaires-utilisateurs/  → Commentaires notables des réseaux sociaux
├── questions-frequentes/       → Questions les plus posées
├── sujets-demandes/            → Sujets explicitement demandés
├── retours-veterinaires/       → Informations issues de vétérinaires
└── idees-videos/               → Idées de vidéos issues de la communauté
```

---

## Règle d'or

> Un commentaire comme *"Mon chien ne mange plus après ses chaleurs"* est une idée de guide.
> Une question comme *"Peut-on donner de la pastille de menthe à son chat ?"* est un sujet de vidéo.

Chaque signal communauté doit être capturé ici avant d'être traité dans le calendrier éditorial.

---

## Processus de capture

1. Lire les commentaires après chaque publication
2. Copier les commentaires / questions notables dans `commentaires-utilisateurs/`
3. Classifier dans la bonne catégorie
4. Lors de la weekly review : convertir en sujets potentiels
5. Ajouter à `guides/idees-futures/` si validé
