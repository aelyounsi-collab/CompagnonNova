# Commentaires Utilisateurs — CompagnonNova

> Archive des commentaires notables issus des réseaux sociaux. Source d'inspiration éditoriale.

---

## Format de capture

```markdown
### [DATE] — [PLATEFORME] — [VIDÉO SOURCE]

**Commentaire** :
> "[texte exact du commentaire]"

**Signal** : [question / témoignage / urgence / demande / incompréhension]
**Sujet potentiel** : [idée de guide ou de vidéo]
**Priorité** : Haute / Moyenne / Basse
**Statut** : [ ] À traiter | [ ] Converti en sujet | [ ] En production | [x] Publié
```

---

## Commentaires capturés

### Exemple — Modèle de saisie

**Commentaire** :
> "Mon chien ne mange plus après ses chaleurs, c'est normal ?"

**Signal** : Question + inquiétude
**Sujet potentiel** : "Chienne en chaleurs : pourquoi elle ne mange plus et quand s'inquiéter ?"
**Priorité** : Haute
**Statut** : [ ] À traiter

---

> Alimenter après chaque séance de lecture de commentaires.
